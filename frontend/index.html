How much storage do you need?

First, please select the amount of storage that you need. Below is a rough guide of how much data you can store with a given capacity.

Storage capability 320GB 500GB 1TB (*1000GB*)
Songs 60,000 125,000 250,000
Photos 60,000 100,000 200,000
Videos 24 hours 38 hours 76 hours

If you are not sure, 1TB should be plenty.

There are three main types of internal storage:

  • HDD - Hard Disk Drives.
    Slow but cheap and offer A LOT of storage capacity.

  • SSD - Solid State Drives.
    So fast that it loads within a blink of an eye! Slightly more expensive than HDD.
    • .
  • SSHD - Solid State Hybrid Drives. (User Choice!)
    The Best of Both World! With HDD and SSD, frequently accessed files are on the Flash memory, so you have speed!
    While the rest are on the hard disk, so you have extra storage.

Select storage type based on your needs. If you are not sure which to pick, let us pick for you (SSHD)!
