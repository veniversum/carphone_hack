If you are not sure, 1TB should be plenty.

There are three main types of internal storage:

  • HDD - Hard Disk Drives. These are the ones that you can find in most laptops and PCs. They contain a rotating disk covered in magnetic material and a moving arm that reads data off the disks. Because of all the mechanical parts they tend to be slow. However, they offer the most storage capacity.
  • SSD - Solid State Drives. As the name suggests, they contain no moving parts and therefore are much faster than HDDs, allowing your computer to start up and load things from memory much faster, which means less waiting! Also, they do not drain battery as much as HDDs. However, the technology is quite new and so the drives usually have less capacity than HDDs with the same price.
  • SSHD - Solid State Hybrid Drives. SSHDs contain both a HDD and an SSD so you can get the best of both worlds. All the files that the computer has to access frequently are on the SSD, while the bulk storage is on the HDD. Nice!

Select storage type based on your needs. If you are not sure which to pick, just go with a HDD!
